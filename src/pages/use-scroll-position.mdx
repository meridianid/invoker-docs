# useScrollPosition

Keep track of window scroll position while the user scrolls. Optionally you can pass in a throttle wrapper to throttle the update calls (e.g. using [lodash.throttle](https://www.npmjs.com/package/lodash.throttle "lodash.throttle") or `window.requestAnimationFrame`).
