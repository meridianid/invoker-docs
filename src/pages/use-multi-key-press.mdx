# useMultiKeyPress

This hook will tell you if a specifig multiple-keys is currently being pressed and return a boolean value indicating that.

## Usage

```jsx live=true
const Hello = () => {
  return (
    <Text heading2>Hello world</Text>
  )
}

render (
  <Hello/>
)
```
